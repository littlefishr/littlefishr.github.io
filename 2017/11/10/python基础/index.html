    输出由1，2，3，4四个数字组成的每位数都不相同的所有三位数

    digits=(1,2,3,4)
 for i in digits:
    for j in digits:
         if i==j:
            continue
         for k in digits:
             if k==i or k==j:
                continue
         print(i*100+j*10+k)

    生成一个含有20个随机数的列表，要求所有元素不相同且值介于1~100

    import random
 x=[]
 while True:
    if len()==20:
        break
    n=random.randint(1,100)
    if n not in x:

    计算组合数C(n,i),即从n个元素中任选i个，有多少选法

    import math
 def C(n,i):
    return int(math.factorial(n)/math.factorial(i)/math.factorial(n-i))
 print (C(6,2))
