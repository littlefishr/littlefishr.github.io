    ubuntu server 安装以及使用相关小结
    2018-04-22

    下面给出我在安装ubuntu server以及使用时的小捷径

    安装

    这里给一个下载iso的链接，其中有desktop和server

    http://mirrors.163.com/ubuntu-releases/16.04/

    安装步骤与安装桌面版的相似。我下载的是16的

    注意：选择语言时，选择中文可能会出现问题

    ubuntu环境下的ssh安装

    过多的不介绍，这里只给出安装步骤

    安装客户端

    ubuntu缺省已经安装了ssh client

    sudo apt-get install ssh

    这里希望你已经安装并使用过Putty。我这里是在使用过kali、

    ubuntu desktop后，想再来尝试一下server版

    ubuntu缺省没有安装ssh server以及ssh配置等在这里不过多介绍

    ubuntu sudo不用输入密码方法

    这里我的用户名是yu（admin组）,我想将admin组的用户sudo都不用密码方法如下：

    运行命令：sudo visudo

    编辑：vi

    更改：你会看到”%admin ALL=(ALL) ALL” 将其改为：

    %admin ALL=(ALL)NOPASSWD:NOPASSWD ALL

    保存：这里是必须要注意的，要用”wq!” 加个叹号强制保存

    Putty登录时，无密码登录（已load ubuntu）

    1.生成公钥和私钥(puttygen)

    这里要注意，你的公钥和私钥要保存在本地，不要弄丢哦

    2.ubuntu中：输入.ssh/authorized_keys,把公钥内容复制到其中

    3.putty中需要设置的两处，添加私钥和用户名：

    SSH——Auth——找到private key file foe authentication——Browse

    Connection——Data——Auto-login username输入你的用户名

    最后到Session保存一下就可以了（注意保存的是你要修改的ubuntu）
