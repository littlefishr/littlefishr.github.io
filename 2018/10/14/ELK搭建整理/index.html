    Ubuntu16、java1.8、elk最新版

    0x01 安装配置Java环境

    下载jdk

    https://www.oracle.com/technetwork/cn/java/javase/downloads/jdk8-downloads-2133151-zhs.html

    创建/usr/java目录

    sudo mkdir /usr/java

    将下载的文件粘贴到/usr/java/目录下，并解压缩

    sudo tar -zxvf jdk-8u181-linux-x64.tar.gz

    修改环境变量

    sudo vim ~/.bashrc

    在文件末尾追加下面内容：

    export JAVA_HOME=/usr/java/jdk1.8.0_181  
# 目录要换成自己解压的jdk目录
export JRE_HOME=${JAVA_HOME}/jre  
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib  
export PATH=${JAVA_HOME}/bin:$PATH

    使环境变量生效

    source ~/.bashrc

    查看是否安装成功。

    java -version

    0x02 ELK安装与配置

    下载ELK组件

    https://www.elastic.co/cn/downloads

    logstash安装配置

    sudo tar -zxvf logstash-6.4.2.tar.gz -C /usr/local/
sudo chmod -R 777 解压后的文件

shipper配置文件
cd /usr/local/logstash-6.4.2/
mkdir conf
cd conf
sudo vim ./shipper.conf

添加一下配置,告诉logstash应该如何处理数据：
input {
  file {
path => "/var/log/apache2/access.log"
start_position => "beginning"
  }
}
#logstash针对apache流量可直接利用的规则
filter {
  grok {
match => { "message" => "%{COMBINEDAPACHELOG}" }
  }
}
output {
  elasticsearch {
  hosts => ["127.0.0.1:9200"]
index => "apache-access-log-%{+YYYY.MM.dd}"
  }
}

运行shipper.conf
./logstash -f 文件所在路径

    elasticsearch安装配置

    sudo tar -zxvf elasticsearch-6.4.2.tar.gz -C /usr/local/
sudo chmod -R 777 解压后的文件
启动
：./elasticsearch(bin目录下)

    kibana安装配置

    sudo tar -zxvf kibana-6.4.2-linux-x86_64.tar.gz -C /usr/local/
sudo chmod -R 777 解压后的文件
vim /usr/local/kibana-6.4.2-linux-x86_64/config/kibana.yml
修改:
server.host: "0.0.0.0"
启动：
./bin/kibana
